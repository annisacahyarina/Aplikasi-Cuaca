**# Aplikasi-Cuaca**

Sebuah aplikasi yang dibangun dengan **Ionic Framework** yang memiliki fitur:

- Geolocation: mengetahui posisi anda saat ini
- Informasi cuaca (khusus wilayah Amerika) : dari Image Server
- Pilihan basemap
